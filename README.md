# Google Books Search

Google Books search with AngularJS

## Install

Install node.js and bower dependencies:

```
npm install
```

## Run

Start server:

```
npm start
```

Use app by opening ```http://localhost:3000/``` in your browser.

## Tests

Run unit tests:

```
npm test
```

Single run of the tests:

```
npm run test-single-run
```
